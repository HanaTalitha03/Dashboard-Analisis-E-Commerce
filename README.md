# **✨Dashboard Analisis E-Commerce Public Dataset**

## **Deskripsi**
Aplikasi dashboard interaktif berbasis Streamlit yang digunakan untuk menganalisis data e-commerce. Dashboard ini menampilkan metrik utama seperti total transaksi, total revenue, serta analisis hubungan antara review score dan revenue produk. Pengguna dapat melakukan eksplorasi data secara dinamis melalui filter rentang waktu dan lokasi (state) untuk mendapatkan insight yang lebih mendalam.

## **Setup Environment (venv)**
```bash
python -m venv venv
venv\Scripts\activate
pip install streamlit pandas matplotlib seaborn
```

## **Run streamlit app**
```bash
streamlit run dashboard/dashboard.py
```
